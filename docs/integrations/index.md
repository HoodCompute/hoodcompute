# Integrations

HoodCompute is designed to slot into your existing AI stack with minimal code changes. The API is OpenAI-compatible, and the SDK provides typed wrappers for the most common patterns.

- [OpenAI-Compatible Usage]({% link integrations/openai-compatible.md %}): use HoodCompute as a drop-in replacement for the OpenAI API
- [JavaScript and TypeScript SDK]({% link integrations/javascript-sdk.md %}): the `@hoodcompute/sdk` package with streaming, React hooks, and on-chain receipts
- [Python]({% link integrations/python.md %}): usage with the OpenAI Python SDK and raw `httpx`
- [LangChain]({% link integrations/langchain.md %}): using HoodCompute as a LangChain LLM
